# 현준이 놀이터 🟡📚

아빠가 만든 아이용 웹앱 모음 (단일 폴더 = 단일 사이트로 호스팅).

- **`index.html`** — 런처(놀이터 선택 화면)
- **`nori/`** — 📚 빨강이 똑똑 놀이터: 한글·숫자·알파벳 학습앱
  - 한글: 배우기 / 글자 만들기 / 받침 글자 / 첫글자 찾기 / 글자 찾기
  - 숫자: 배우기 / 숫자 찾기 / 더하기 놀이
  - ABC: 배우기(대·소문자) / ABC 찾기 / 소문자 찾기
- **`pacman/`** — 🟡 빨강이 팩맨 게임

각 앱은 PWA(홈 화면 추가 + 오프라인 캐시)로 동작하고, 글을 못 읽는 아이를 위해
모든 지시·칭찬을 음성(Web Speech API)으로 읽어줍니다.

## 로컬에서 보기
```sh
cd <이 폴더>
python3 -m http.server 8424
# http://localhost:8424  →  런처
# http://localhost:8424/nori/nori.html  →  학습앱
```

> 음성·효과음은 첫 화면 터치 후 동작합니다(브라우저 자동재생 정책).
> 안드로이드/아이패드에서 한국어 발음이 안 나오면, 기기 설정에서 **한국어 TTS 음성**이
> 설치돼 있는지 확인하세요.
